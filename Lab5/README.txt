In order to run the program, type the following in the terminal:
   java -jar CpGIsland.jar

Press the `Find File` button to bring up the file browser. Navigate to the FASTA or
batch file containing the DNA sequence. Press `Find File` to load the file into the software.

Press `Go` to generate the CSV containing CpG island information.

A dialog will pop up telling you the CSV file's location (this will be the same location as the test case).

To compile:
   javac *.java
To run from compilation:
   java CpGIsland
